Welcome to a Sample Page!

This is a basic HTML page demonstrating the core structure.

Key elements:

  • The **<head>** section contains meta-information like the page title and character set.
  • The **<body>** section contains all the visible content of the page.
  • Headings (**<h1>**, **<h2>**, etc.) structure the content.
  • Paragraphs (**<p>**) are used for blocks of text.
  • Unordered lists (**<ul>**) are used for bullet points.
