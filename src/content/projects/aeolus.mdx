# Overview

Aeolus is a hybrid thrust vector control (TVC) & aerodynamic fin controlled electric ducted (EDF) fan rocket designed for controlled flight in a small space. Unlike the common solid rocket motors, an EDF can be throttled continuously, enabling controlled ascent, hover, and landing without the large field clearances that these motors require. This throttleability introduces a harder engineering problem: at hover, airspeed is zero, fins generate no restoring moment, and the vehicle is aerodynamically unstable. The only stabilization mechanism is active attitude control via four exhaust vanes in the EDF jet — which requires accurate real-time attitude estimation, a fast closed-loop controller, and a reliable telemetry link, all running on custom flight hardware. Every subsystem was designed and built from scratch: the sensor fusion pipeline, radio protocol, RPM governor, 6DOF simulation, and ground station.
