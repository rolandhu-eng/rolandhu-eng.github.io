# over
This documentation covers the development of the discrete attitude control loop designed for the subsonic wind tunnel project. The system utilizes a PID controller architecture to maintain stable pitch angles during high-flow test phases.

# Control Law
The controller operates on a discrete time-step $\Delta t$, calculating output based on the error $e(t)$ between the target setpoint and current sensor reading:

$$u(t) = K_p e(t) + K_i \int_{0}^{t} e(\tau) d\tau + K_d \frac{de(t)}{dt}$$

# Implementation
The attitude logic is implemented within a MATLAB S-Function block to interface directly with Simulink's solver. 
## code
```matlab
function [sys,x0,str,ts,simStateCompliance] = sfun_attitude(t,x,u,flag)
    % Initialize system sizing parameters
    switch flag
      case 0
        [sys,x0,str,ts,simStateCompliance] = mdlInitializeSizes();
      case 3
        % Compute outputs based on current time and state
        sys = mdlOutputs(t,x,u);
      case 9
        % Terminate simulation gracefully
        sys = mdlTerminate(t,x,u);
      otherwise
        DAStudio.error('Simulink:blocks:unhandledFlag', num2str(flag));
    end
end
```
